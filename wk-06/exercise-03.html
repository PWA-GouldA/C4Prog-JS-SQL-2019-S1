Week 06 - Exercise 03

Simple Grocery Store

Practice manipulating the DOM

Problem to solve

This exercise builds on Week 05's exercise 3.

Create a JavaScript script that:

  1. Asks the user to select a fruit or vegetable from a drop down (select) box with the options fruit (apple, cherry, banana, orange, date, kiwi fruit, pear, peach, apricot) and vegetables (potato, beetroot, carrot, turnip, swede, celery, fennel)
  2. Asks the user for the price of the fruit/vegetable per kilogram
  3. Asks the user for the quantity (kilograms) of the fruit/vegetable
  4. Calculates the total cost of the fruit/vegetable (price per kilogram times quantity of fruit/vegetable)
  5. Displays the fruit/vegetable name, the price, the quantity and total cost

When the user makes an error the page needs to prompt them to correct it. The page must check for the following errors and display them to the user:

  1. Name of Fruit/Vegetable: if the name is not entered, then prompt the user to select a fruit or vegetable
  2. Price per Kilogram: if the user enters a price that is less than 0 then they are told to enter a positive number for the price per kilogram.
  3. Price per Kilogram: if the user enters a price that is greater than $250 then they are told to enter a price that is less than $250 per kilogram.
  4. Quantity: if the user enters a quantity that is less than 0 then they are told to enter a value that is greater than 0.
  5. Quantity: if the user enters a quantity that is greater than 1000 then they are told to enter a value that is less than 1000.

When an error is encountered, the results are should be blanked out.

Make the output similar to this:

  • Product: PRODUCT NAME
  • Price/Kg: PRICE
  • Quantity (Kg): QUANTITY
  • Total: $ TOTAL
