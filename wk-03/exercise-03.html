Problem to solve

Create a JavaScript script that:

  1. set a counter to zero
  2. Asks the user to enter a number
  3. while the total entered is less than 100:
    1. Add the number to the total
    2. add one to the count
    3. Asks the user to enter another number
  4. Calculate the mean (average) of all the numbers entered
  5. Display the count, total and average in an alert
