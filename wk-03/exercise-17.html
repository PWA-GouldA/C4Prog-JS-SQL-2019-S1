Problem to solve

Create a JavaScript script that:

  1. create a variable called factorial, and set to one
  2. create a variable called number and set it to zero
  3. ask the user to enter an integer between 1 and 30, putting the integer into the variable number
  4. convert number to an integer (parse int)
  5. for count is from 1 to number:
    1. factorial equals factorial times count
  6. Display number! is factorial

The output will be similar to:

5! is 120
