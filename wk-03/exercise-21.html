Problem to solve

Write a JavaScript program that plays Fizz-Buzz

It will play the game from 1 to 20 displaying:

  1. Fizz if the number is divisible by 2
  2. Buzz if the number is divisible by 3
  3. The number if it is not divisible by 2 or 3

The fist six results will be: 1, Fizz, Buzz, Fizz, 5, Fizz-Buzz...
