Problem to solve

Create a JavaScript script that:

  1. set a counter to two
  2. while the counter is less than ten:
    1. if the counter is odd then: [hint: think modulo/remainder]
      1. output to the console "ODD"
    2. else:
      1. output the counter to the console
    3. increment the counter by one
  3. End while
