Problem to solve

Create a JavaScript script that:

  1. Asks the user for a temperature in Celcius
  2. Calculates the temperature in Fahrenheit as Celcius * 9 / 5 + 32
  3. Calculates the temperature in Kelvin as Celcius + 273.15
  4. Displays the temperature in Celcius, Fahrenheit and Kelvin

Make the output similar to this:

-40F is -40C or -233.15K
