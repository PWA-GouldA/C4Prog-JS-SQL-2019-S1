Problem to solve

Create a JavaScript script that:

  1. create a variable called leapYear, and set to false
  2. create a variable called theYear, and set to 0
  3. Ask the user for the year
  4. If the year is divisible by 4 then:
    1. set leap year to be True
  5. If the year is divisible by 100 then:
    1. set leap year to be False
  6. If the year is divisible by 400 then:
    1. set leap year to be True
  7. Alert the user to if the year is a leap year or not.

The alert should display the result in a manner similar to:

1999 is not a leap year.
2000 is a leap year.
