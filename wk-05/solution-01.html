Converts from Fahrenheit to Celcius

Create a form that accepts a temperature in Fahrenheit and converts it into Celcius, displaying the result in a result area.

Create the HTMl form on this page that:

  1. Displays a single input field that accepts a temperature
  2. Displays a submit button

Create an HTML result area on this page, below the form, that:

  1. Displays "Celcius" in one column
  2. Displays the converted temperature in a second column

Create the HTMl form and the JavaScript that:

  1. Gets the value from the input field
  2. Converts it into a real / floating point number
  3. Calculates the temperature in Celcius (C = (F-32)*5/9)
  4. Displays the result in the Celcius result column
