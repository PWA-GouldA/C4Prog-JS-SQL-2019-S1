Week 05 - Exercise 02

Calculate and Display Factorials

Create a form that accepts an INTEGER number and on submitting calculates the factorial of the number.

Create the HTMl form on this page that:

  1. Displays a single number input field that accepts an integer
  2. Displays a submit button

Create an HTML result area on this page, below the form, that:

  1. Displays "x!" in one column, where the x is replaced by the integer the user entered
  2. Displays the calculation for the x! (eg: 5! would display 5 * 4 * 3 * 2 * 1 or 1 * 2 * 3 * 4 * 5)
  3. Displays the actual result in a new row.

Create the JavaScript that:

  1. Gets the value from the input field
  2. Converts it into an integer number
  3. Calculates number factorial
  4. Shows the required results in the form
5! =
1 * 2 * 3 * 4 * 5
=
120
