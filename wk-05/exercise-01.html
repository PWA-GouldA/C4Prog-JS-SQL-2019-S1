Problem to solve

Create a JavaScript script that:

  1. Asks the user for the name of a fruit
  2. Asks the user for the price of the fruit per kilogram
  3. Asks the user for the quantity (kilograms) of the fruit
  4. Calculates the total cost of the fruit (price per kilogram times quantity of fruit)
  5. Displays the fruit, the price, the quantity and total cost

Make the output similar to this:

Apple: 5.3kg at $3.30 = $17.49
