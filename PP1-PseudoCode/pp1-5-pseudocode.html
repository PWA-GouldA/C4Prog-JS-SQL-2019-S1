Using the provided HTML and JS files, create the JavaScript that:

  • Gets a number from a text box and the choice of Celcius or Fahrenheit from a radio button and displays the temperature in the opposite scale.
  • The input and the output will be from an HTML based form (input) and paragraph (output).
  • If the user enters anything that is not a number then an error message is displayed “Error, you did not enter a number”.

You will write and use your own functions to:

  • convert from Fahrenheit to Celcius
  • convert from Celcius to Fahrenheit
    

        let doc equal the document

        let temperature input equal the doc's get element by id for the temperature text box (HTML ID)
        let convert button equal the doc's get element by id for the convert button (HTML ID)
        let converted output equal the doc's get elements by id for the output zone (HTML ID)

        function button pressed ( event )
            event prevent default
            temp equals temperature input's value
            convert choice equals convert to radio's value

            if is a number (temp) then
                if convert choice is F then
                    result equals fahrenheit to celcius ( temp )
                else
                    if convert choice is C then
                        result equals celcius to fahrenheit ( temp )
                    end if
                end if

                converted output's inner text equals result
            end if
        end function


        function fahrenheit to celcius ( temperature )
            return ( temperature minus 32 ) times 9 divide by 5
        end function


        function celcius to fahrenheit ( temperature )
            return ( temperature times 5 divided by 9 ) plus 32
        end function
