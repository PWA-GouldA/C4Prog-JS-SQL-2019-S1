Create a JavaScript script that displays on a web page the prime numbers between 2 and 32767, in 5 columns.

                output equals output plus "<div class='row'>";
            end if more numbers
        end if counter is divisible by max columns with no remainder
    end if prime
end for num

end if more columns needed
