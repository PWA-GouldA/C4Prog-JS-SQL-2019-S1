PP1 Q6 Pseudocode

Create a simple grocery store price calculator that:

  1. 1. Asks the user for the name of a fruit (from a combo box)
  2. 2. Asks the user for the price of the fruit per kilogram (text box)
  3. 3. Asks the user for the quantity (kilograms) of the fruit (text box)
  4. 4. Calculates the total cost of the fruit (price per kilogram times quantity of fruit)
  5. 5. Displays the fruit, the price, the quantity and total cost

Make the output similar to this:

Product

PRODUCT NAME

Select a product

Price/Kg

PRICE

Enter a value that is 0 or greater

Quantity (Kg)

QUANTITY

Enter a value that is 0 or greater

Total $

TOTAL

Where the second column shows the actual values selected.

The calculation should update every time a value for the product, price or quantity is changed.

If a value that is invalid then there should be no result, but an error message displayed. The error messages are shown in the test data.

The Totals should be displayed to two decimal places only.
