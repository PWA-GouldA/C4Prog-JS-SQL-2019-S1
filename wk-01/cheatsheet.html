Tables

The table element has a number of components including thead, tfoot, tbody, th, td, tr and caption.

Table Caption
Column 1 Column 2 Column 3
Item 1/1 Item 2/1 Item 3/1
Item 1/2 Item 2/2 Item 3/2
Footer 1 Footer 2 Footer 3
  • table is the wrapper for the table
  • tr is a row in the table
  • td is a cell in the table
  • th is a "heading" cell in the table
  • thead contains header for the table
  • tbody contains body content of the table
  • tfoot contains footer for the table
  • caption displays a caption at the top of the table
